# REALMENT drug prediction pipeline

This repository contains the scripts and key resources for a the development of a drug prediction pipeline, as part of REALMENT [grant details to be added].

The pipeline relies on genome-wide genetic data from the UK Biobank (UKB) together with tissue-specific eQTL data from GTEx and drug-induced transcriptomic signatures from LINCS2. With this, we first impute the genetically regulated component of gene expression (GREx) across different brain and blood tissues for each individual using [LAVA](https://github.com/josefin-werme/LAVA). Individual level drug response is then predicted by performing expression-signature matching based on imputed GREx profiles and drug-induced transcriptomic signatures from LINCS2. 

A key aim of the project was to determine the impact of different parameter choices — including SNP filtering thresholds, tissue selection, and signature matching method — on prediction accuracy for different drugs.

---

## Pipeline overview

1. **Filter GTEx eQTLs** by significance threshold to define the SNP sets used for imputation (either all SNPs, or subsetted by Z-scres)
2. **Impute GREx** — for each individual in the UKB cohort, impute the genetically regulated component of gene expression across all available genes and tissues
3. **Predict drug responses** — match individual GREx profiles against LINCS2 drug-expression signatures to generate ranked drug predictions per individual
4. **Validate predictions** — evaluate whether predicted drugs are enriched for the drugs participants are actually prescribed, using ROC analysis and enrichment testing against UKB medication data

## Repository structure

| Directory | Description |
|---|---|
| `scripts/` | Pipeline scripts; see `scripts/README.md` for details |
| `programs/` | Custom LAVA_batch_locus R package used for GREx imputation |
| `data/lava_gtex/` | Processed GTEx eQTL data; see `data/lava_gtex/README.md` |
| `data/signatures/` | LINCS2 drug-expression signatures; see `data/signatures/README.md` |
| `data/ukb_phenotypes/` | Relevant medication and health-related data extracted from UKB, including prescription status for ATC drug classes of interest and relevant diagnoses |
| `grex_imputed/` | Output: imputed per-individual GREx values (generated by pipeline) |
| `drugs_pred/` | Output: per-individual drug predictions, in dirs named `{maf}-{eqtl_z_thresh}-{tissue}-{method}` |
| `validation/` | Output: ROC curves and enrichment results (same naming convention) |
